# # ☕ Café Finder
Café Finder is a simple web application that helps users find nearby cafés in the city and get directions using an interactive map.

Built with Flask (backend) and Leaflet.js (frontend), it demonstrates how to integrate location search, geocoding, and routing APIs into a real-world project.

#🔹 Features

🌍 Search cafés across the whole city

📍 Find cafés near your current location

🗺️ Interactive map with café markers

🧭 Step-by-step routing directions

🎨 Custom styled map using OpenStreetMap + Leaflet

# 🛠️ Tech Stack

Backend: Flask (Python)

Frontend: HTML, CSS, JavaScript

Maps & Routing: Leaflet.js, OpenStreetMap, OpenRouteService API
